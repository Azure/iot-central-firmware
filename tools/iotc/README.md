### Azure IOT - ARM Mbed - Easy build

ARM mbed compiler tools - a wrapper around ARM mbed-cli

![AzureIOT ARM mbed easy build](azureiot_mbed.gif)

#### Requirement

Install Docker for your system.

#### How To

##### Install

```
npm install -g @azure-iot/mbedc
```

##### Use

```
mbedc --help
```
